# Waikiki Dental

Modern Next.js redesign for Waikiki Dental in Roseville, CA.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- lucide-react icons
- Vercel deployment

## Local Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```

## Source Notes

The site preserves Waikiki Dental's public content facts: Roseville address,
phone number, Dentrix booking link, doctor bio, service menu, new-patient
information, hours, and testimonials. The live `waikikidental.com` domain is not
connected in v1; this deploy ships to a Vercel URL first.
